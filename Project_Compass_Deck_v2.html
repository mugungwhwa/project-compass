A1

Retention Prediction Engine

The Five Retention Properties • Dual-layer prediction architecture • Bayesian Prior→Posterior

The Five Mathematical Properties of Retention
P1

Monotonic Non-Increase

D(N) ≤ D(N-1) always

P2

Sequential Dependency

Each day influenced by prior state

P3

Temporal Decrease

Retention declines over time

P4

Decelerating Decline

Rate of decrease slows

P5

Asymptotic Floor

Curve converges to stable base

Dual-Layer Architecture
Layer 1: Parametric (Early)

Power Law + Genre Prior

R(t) = a × t^b + c — works from D1-D7 data. Genre-stratified benchmark bands provide Bayesian prior.

Prior(genre) + Observed(D1-D7) = Posterior
Layer 2: LSTM (Scale)

Deep Learning + Attention

2× LSTM (128 units) + Attention + MC Dropout. Activates at D14+ with sufficient cohort volume.

D14+ sequences → D60-D365 P10/P50/P90
